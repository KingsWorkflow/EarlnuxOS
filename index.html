Project Overview

EarlnuxOS is a custom-built, lightweight operating system designed for the i686 architecture. Built from the ground up in C and Assembly, it focuses on modern kernel concepts like virtual memory management, multi-user security gateways, and a flexible virtual filesystem (VFS).

Kernel Features

• Multi-User Security with Password Masking
• Advanced VFS (ramfs, procfs, vibefs)
• Dynamic Shell with Arithmetic Evaluation
• PCI Bus Discovery & Hardware Probing

Networking

• Integrated TCP/IP Stack
• DHCP Support (Automatic IP Config)
• DNS Resolution Support
• PCI-based Interface Management

Virtual Machine Setup Guide

Ready to run EarlnuxOS locally? Follow these steps to set up the ISO in your favorite VM:

VirtualBox / VMWare

1. Create a New VM
2. OS Type: Other (32-bit)
3. Memory: 128MB (Minimum)
4. Storage: Attach os.iso to the Optical Drive.

QEMU (Linux/macOS/Win)

Run the following command in your terminal:
qemu-system-i386 -cdrom os.iso
